# Ibn Abi Shaybah Collections

Welcome to the Ibn Abi Shaybah Collections project! This repository hosts a web-based version of the classical Islamic hadith compilation by Abu Bakr Muhammad ibn Abi Shaybah (d. 235 AH).

The goal of this project is to make these historical texts accessible online in a structured, searchable, and user-friendly format for students, researchers, and anyone interested in Islamic studies.
